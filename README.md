approval-ui
    <p>A simple UI Toolkit built on Vue.js and iview.</p>

## Install

Using npm:
```
npm install approval-ui --save
```
Usage in Vue
```
import approval from 'approval-ui'
Vue.use(approval)
```

You can find more info [on the website](hhttps://github.com/HappyPot/approval-ui).
